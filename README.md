# System Health Monitor Dashboard
#### A real-time terminal dashboard built with Python that monitors your system's vital statistics live.

## Features:

* Monitor CPU, RAM, Disk, Network usage and UpTime
* Clean dashboard inside terminal using rich
* Desktop notifications for CPU and RAM heavy use
* Logs in a txt file

## Prerequisites
* Open Hardware Monitor

## Install

You can install the .exe from github. You can find the .exe in our project github release.

## Quick Start
1. Run the Open Hardware Monitor
2. After download the .exe open it and monitor your system.
